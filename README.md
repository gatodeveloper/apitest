# API Express


Install dependencies:

```bash
npm install
```


Run the development server:

```bash
npm run dev
```

## Customer endpoints:

eg: example local server http://localhost:3000

### Get All Customers:

```bash
GET /api/v1/customers
```

### Get by Id:

```bash
GET /api/v1/customers/1
```

### Get by name:

```bash
GET /api/v1/customers?search=diego
```
